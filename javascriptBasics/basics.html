JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. Javascript is extremely popular for a variety of reasons. It is a small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. Javascript lives inside a host environment (a web browser or Node server); it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable Declaration: Javascript variables are containers for storing data values. Imagine a cup filled with coffee -- the variable is the cup, the data is the coffee. All JavaScript variables must have unique names (called "identifiers").
  • Variable Assignment: Assignment operators assign values to JavaScript variables -- coffee is poured into the cup, and now it holds a content (a value). The "=" operator is used to make these assignments.
  • Data Types: Data types are an important concept; you cannot operate on a variable if you don't know its data type. JavaScript has six primitive data types: Boolean (true or false), Null i.e. nothing, Number (42, 3.14159), String ("hello world"), Array ([1,2,3,4]), and Object ({first: "Jane", last: "Doe"}).
  • Conditonals: Conditionals designate a block of code to be executed if (and, unless the block occurs or is referenced elsewhere, only if) a certain condition is met. Supplemental conditionals like else if and else then inform the program what to do when the condition is not met.
  • Loops: There are many different kinds of loops in every programming language, but they all essentially do the same thing: they repeat an action a given number of times or until a certain condition is met.
  • Functions: functions are like variables associated not with more or less simple values but blocks of code. When the function is "called," the block of code is executed. Functions allow developers to write code that is both precise and clear: storing code keeps us from repeating ourselves, and labeling it helps other people to understand what the code does.
